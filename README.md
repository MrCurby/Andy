# Andy — Übersicht über wiederkehrende Ausgaben

Kurzbeschreibung
----------------
`Andy` ist eine Blazor-Anwendung (Ziel: .NET 10), die dir eine kompakte Übersicht über wiederkehrende Ausgaben liefert — z. B. Abos, Verträge und Wohnen/Nebenkosten. Die UI ist auf schnelle Navigation und Wiederkehrendes (Subscriptions) ausgelegt.

Zweck
-----
- Übersicht und Verwaltung von wiederkehrenden Zahlungen (Abonnements, Verträge, fixe Nebenkosten).
- Erkennen von Summen, Fälligkeiten und Optimierungspotenzialen.
- Grundlage für Analysen und Erinnerungsfunktionen.

Voraussetzungen
--------------
- .NET 10 SDK
- Visual Studio 2026 oder `dotnet` CLI
- Git
